# Screenshots Folder

Place your app screenshots here. Recommended naming convention:

```
lingocast.png          (or .jpg, .webp)
explores-berlin.png
examcurrator.png
mindflow.png
dataflow.png
```

## Usage in Code

Reference in your components like:

```tsx
screenshot: "/screenshots/lingocast.png"
```

## Recommended Image Specs

- **Format**: WebP (best compression) or PNG
- **Resolution**: 1200x800px or 1920x1080px
- **Aspect Ratio**: 3:2 or 16:9
